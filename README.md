# DeveloperHub

DeveloperHub es una API desarrollada con **.NET 8**, diseñada para facilitar la colaboración entre desarrolladores a través de la creación de proyectos, comentarios y perfiles integrados con GitHub y Discord.

## 🚀 Características

- Registro e inicio de sesión con autenticación JWT
- Creación y gestión de proyectos colaborativos
- Sistema de etiquetas (tags) para categorizar proyectos
- Comentarios en proyectos
- Perfil de usuario con soporte para GitHub y Discord
- Paginación en endpoints de listado
- Arquitectura limpia (Clean Architecture)
- Validaciones con FluentValidation
- DTOs y AutoMapper
- Control de acceso basado en el usuario propietario

## 🧱 Tecnologías

- .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- AutoMapper
- FluentValidation
- JWT Bearer Authentication
